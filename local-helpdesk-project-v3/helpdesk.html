Настройка общего склада

Для синхронизации остатков картриджей между всеми пользователями используется JSONBin.io — бесплатный облачный JSON-хостинг.

  1. Зарегистрируйтесь на jsonbin.io
  2. Создайте новый Bin с содержимым: {}
  3. Скопируйте BIN ID и X-Master-Key
  4. Вставьте их ниже и нажмите Сохранить
